# farmer_labour_app
# 🌾 Farmer Labour Connect

A Flutter-based platform designed to connect farmers with labourers for agricultural work management, hiring, and communication. The application simplifies the process of finding labourers for farming activities and improves coordination between farmers and workers.

---

## 🚀 Features

* 👨‍🌾 Farmer Registration & Login
* 👷 Labour Registration & Profile Management
* 🔍 Search and Hire Labourers
* 📍 Location-based Work Opportunities
* 💬 Communication Between Farmers & Labourers
* 🔐 Authentication using Supabase
* ☁️ Real-time Database Integration
* 📱 Responsive Mobile UI using Flutter

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to project directory

```bash
cd Farmer-Labour-Connect
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
